Fake Recurly
============

This is a really quick and dirty replica of the recurly API that I use for
integration tests. I've only added functionality as I needed it, so don't expect
much.

Install
-------

```
# virtualenv .
# . bin/activate
# pip install -r requirements.txt
```

Usage
-----

```
# python app.py
```
